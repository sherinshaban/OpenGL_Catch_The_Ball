# Catch The Ball Game 

A simple 2D interactive game developed using **C++** and **OpenGL (GLUT)**. The objective is to catch the falling balls using a basket/paddle controlled by the player.

##  Game Mechanics
* **Objective:** Catch as many falling objects as possible to increase your score.
* **Game Over:** The game ends (or score decreases) if you miss a ball.
* **Graphics:** Built using OpenGL primitives (circles, quads) and transformation matrices.

##  Controls
* **Move Left:** [Left Arrow Key] or [A]
* **Move Right:** [Right Arrow Key] or [D]
* **Exit:** [Esc]
